Class MonchBuffer

The superfast buffer class that's the main part of Monch.

Here's a simple example to try out:

const buffer = new MonchBuffer()
buffer.grow(8)
buffer.writeUint32BE(0x00, new Uint32Array([ 0xDEADBEEF ]))
console.log(buffer.toArray())
